Phase 1 — Active

KOSL Albu

Egg white photoresist for home semiconductor fabrication

Open source process for lithography using egg white (albumen) as photoresist with riboflavin (vitamin B2) as photosensitizer. Uses a 5W 450nm laser diode on a $525 CNC setup. Develops in plain water.

Currently validating photoresist at 100μm feature size. The ovalbumin protein crosslinks under 450nm illumination via riboflavin-mediated singlet oxygen generation, creating water-insoluble patterned regions.

100μm target $525 setup 450nm laser Water develop
Planned

KOSL Machine 3

Automated maskless direct-write lithography

Desktop lithography system with machine vision alignment and automated wafer handling. 10 micrometer resolution using DLP maskless projection. Pending completion of Albu Phase 2 validation.

Will serve as the platform for the KOSL 10MT5000 Auto product line — a turnkey desktop semiconductor fab system with HEPA enclosure.

10μm target Machine vision DLP projection
Roadmap

Albu development phases

From photoresist validation to a fabricated CPU — the complete research arc.

Phase 1 — Current
Photoresist Validation
Prove egg white photoresist works at 450nm using a CNC laser and optical microscope. Success criterion: microscope-verified patterns at 100μm feature size on glass, developed in water.
$525 setup CNC 3018 + laser
Phase 2
First Transistors
Build first working transistors — ZnO TFTs on glass, Schottky-barrier MOSFETs, laser-doped Si MOSFETs, Cu2O p-type TFTs. Culminates in a 5-stage ring oscillator (10 transistors).
ZnO TFT Si MOSFET Ring oscillator
Phase 2.5
Automated Lithography Upgrade
Transition from CNC laser to DLP maskless projection lithography. Semi-automated spin coating and development. Basis for the KOSL 50MT500 Auto product.
50μm features DLP projection
Phase 3
Full Automated Fab Line
Complete automated fabrication process — precision stages, HEPA enclosure, machine vision alignment. 10 engineering sub-phases. Basis for the KOSL 10MT5000 Auto.
10μm features ~5,000 transistors
Phase 4
First CPU
Fabricate a working CPU — either a custom 4-bit "Albu-4" (~910 transistors) or a SERV RISC-V core (~4,500 transistors), depending on yield data from Phase 3.
4-bit or RISC-V Up to 4,500 transistors
Future
DUV Stepper
Schwarzschild reflective objective DUV stepper using a surplus KrF excimer laser. 500nm resolution — enabling Pentium-class chip complexity (millions of transistors).
500nm features Millions of transistors
